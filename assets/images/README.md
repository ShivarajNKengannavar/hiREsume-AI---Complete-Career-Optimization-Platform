# Images Directory

This directory contains images for the hiREsume project.

## Recommended Images:

### Logo and Branding
- `logo.png` - Main application logo
- `favicon.ico` - Website favicon
- `banner.jpg` - Hero section banner

### Screenshots (place in ../screenshots/)
- `dashboard.png` - Main dashboard view
- `resume-maker.png` - Resume creation interface
- `job-search.png` - Job search functionality
- `portfolio-preview.png` - Portfolio generator
- `analytics.png` - Analytics dashboard

### Icons
- `features/` - Feature-specific icons
- `social/` - Social media icons
- `ui/` - UI element icons

## Image Guidelines:
- Use WebP format for better compression
- Maintain consistent aspect ratios
- Optimize for web (max 1MB per image)
- Use alt text descriptions for accessibility
